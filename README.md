## 🎯 Propósito deste Repositório

Diferente de um projeto final ou de um produto em produção, este repositório atua como o meu **Caderno de Rascunhos Tecnológico**. 

Aqui eu registro a minha evolução contínua, guardo trechos de código úteis (*snippets*), testo novas lógicas e resolvo os desafios propostos nas aulas. Este repositório é a prova do meu "código de todo dia" e do meu compromisso com o aprendizado contínuo.

## 📂 Estrutura de Organização

O conhecimento está dividido de forma modular para facilitar a consulta futura e demonstrar organização:

- 📁 `modulo-01-frontend/`: Exercícios de interface, marcação e estilização (HTML, CSS, JavaScript e TypeScript).
- 📁 `modulo-02-backend/`: Lógica de servidor, criação de rotas e construção de APIs.
- 📁 `modulo-03-banco-de-dados/`: Modelagem de dados, consultas SQL e integrações.

## 💡 Como Navegar

Cada pasta de módulo contém subpastas com o nome da aula ou do desafio específico. Dentro delas, você encontrará o código executável e a resolução das atividades passo a passo.
